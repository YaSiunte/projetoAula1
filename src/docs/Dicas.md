1º Abrir o VSCODE
2º No terminal digitar: cd desktop
3º No terminal dentro do desktop: mkdir nomedapasta
4º No terminal: cd nomedapasta
5º Dentro da pasta nomedapasta: code .
6º No terminal da pasta nomedapasta: touch index.html
7º No terminal: mkdir src 
8º No terminal: mkdir css
9º No terminal: mkdir js
10º No terminal: mkdir js
11º No terminal: cd css
12º No terminal: touch estilo.css
13º No terminal: cd ..
14º No terminal: cd js
15º No terminal: touch script.js
16º No terminal: cd ..
17º No terminal: cd .. (ate chegar na raiz do projeto)
18º No terminal: git init
19º Se estivar na FIAP: git config --global user.name "Yasmin"
20º Se estivar na FIAP: git config --global user.email yasiunte@gmail.com
21º No terminal: git status
22º Se estiver vermelho: git add .
23º Se estiver verde: git commit -m "Mensagem do que foi feito"
24º Entrar no github em repositórios, criar um novo repositorio, colocar o nome e deixar publico, clicar no botao criar repositorio
25º No bloco que estiver as 3 linhas:
  1º linha - conecta seu gitbash com seu github
  2º linha - muda o nome master para main 
  3º linha - pega tudo que estiver no projeto e sobe para o github
26º Caso voce esteja em outro pc: no seu repositorio no botao verde(code), copie o link
27º Abra seu vscode no terminal cd desktop: git clone (copie o link), isso vai clonar seu projeto e pode continuar de onde parou
28º Se modificar seu código faça todo o processo do git status
